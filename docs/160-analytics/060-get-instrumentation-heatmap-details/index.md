Title: GetInstrumentationHeatmapDetails (GET /:login/analytics/instrumentations/:id/value/heatmap/details)
---
Text:

Allows you to retrieve the bucket details for a heatmap.

### Inputs

Takes all the same parameters as
[GetInstrumentationHeatmap](#GetInstrumentationHeatmap), and additionally:

||**Field**||**Type**||
||x||Number||
||y||Number||

### Returns

The returned value includes:

||**Field**||**Type**||
||bucket\_time||Number||
||bucket\_ymin||Number||
||bucket\_ymax||Number||
||present||Object||
||total||Number||
